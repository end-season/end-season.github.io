# GUI程序结构与运行机制

* 使用Qt Creator创建一个Widget项目，项目目录树如下：

  ![](/img_post/Qt6_1.1.png)

​	其中，`widget.ui`文件是UI文件，用于窗口界面可视化设计。

## 项目构建

创建项目时，若使用的是`qmake`构建系统，则会生成后缀为`.pro`文件。若使用`cmake`构建系统，则会生成`CMakeLists.txt`文件。

## UI文件

后缀为`.ui`的文件，双击UI文件会打开Qt Designer对窗口界面进行可视化设计。

UI文件经过用户界面编译器（user interface compiler, UIC）编译后会生成`ui_widget.h`，这个文件定义了一个类，用`c++`语言描述UI文件中界面组件的属性、布局以及信号与槽的关联等内容。

* 看以下示例：

  使用Qt Designer为`ui`文件添加一个标签，并输入内容：`hello qt`:

  ![](/img_post./Qt6_1.2.png)

  运行后，打开项目目录下类似这样的路径：`.\build\Desktop_Qt_6_10_2_MinGW_64_bit-Debug\Qt6_1_autogen\include`，就可以找到由UI文件生成的`.h`文件。打开这个文件：

  ```c++
  /********************************************************************************
  ** Form generated from reading UI file 'widget.ui'
  **
  ** Created by: Qt User Interface Compiler version 6.10.2
  **
  ** WARNING! All changes made in this file will be lost when recompiling UI file!
  ********************************************************************************/
  
  #ifndef UI_WIDGET_H
  #define UI_WIDGET_H
  
  #include <QtCore/QVariant>
  #include <QtWidgets/QApplication>
  #include <QtWidgets/QLabel>
  #include <QtWidgets/QWidget>
  
  QT_BEGIN_NAMESPACE
  
  class Ui_Widget
  {
  public:
      QLabel *label;
  
      void setupUi(QWidget *Widget)
      {
          if (Widget->objectName().isEmpty())
              Widget->setObjectName("Widget");
          Widget->resize(800, 600);
          label = new QLabel(Widget);
          label->setObjectName("label");
          label->setGeometry(QRect(270, 220, 171, 61));
          QFont font;
          font.setFamilies({QString::fromUtf8("Microsoft Tai Le")});
          font.setPointSize(25);
          label->setFont(font);
  
          retranslateUi(Widget);
  
          QMetaObject::connectSlotsByName(Widget);
      } // setupUi
  
      void retranslateUi(QWidget *Widget)
      {
          Widget->setWindowTitle(QCoreApplication::translate("Widget", "Widget", nullptr));
          label->setText(QCoreApplication::translate("Widget", "hello qt", nullptr));
      } // retranslateUi
  
  };
  
  namespace Ui {
      class Widget: public Ui_Widget {};
  } // namespace Ui
  
  QT_END_NAMESPACE
  
  #endif // UI_WIDGET_H
  
  ```

  它做了如下工作：

  * 定义了一个类`Ui_Widget`，用于封装可视化设计的界面。

    > 注意，`Ui_Widget`类没有父类，不是从QWidget继承而来的，也就不是一个窗口类。

  * 为界面中的每个组件定义了一个指针变量，以便窗口类访问窗口中的组件：

    ```c++
    //修改label中的文字
        ui->label->setText("hello world");
    ```

  * 定义了一个函数`setupUi()`，传入`QWidget`类型指针，为传入的widget设置属性，并创建组件，设置其属性:
  
    ```c++
    if (Widget->objectName().isEmpty())
                Widget->setObjectName("Widget");
            Widget->resize(800, 600);
            label = new QLabel(Widget);
            label->setObjectName("label");
            label->setGeometry(QRect(270, 220, 171, 61));
            QFont font;
            font.setFamilies({QString::fromUtf8("Microsoft Tai Le")});
            font.setPointSize(25);
            label->setFont(font);
    ```
  
    接着调用了函数`retranslateUi()`，这个函数设置了界面上各组件的文字属性，如标签的文字、按钮的文字、窗口的标题等。
  
    最后调用`QMetaObject::connectSlotsByName(Widget)`，这个是设置槽函数的关联方式，用于将窗口上各组件的信号与槽自动连接，这里没有信号与槽的使用，因此先不做说明。
  
  * 定义了名称空间`Ui`并定义一个从`Ui_Widget`继承的类`Widget`：
  
    ```c++
    namespace Ui {
        class Widget: public Ui_Widget {};
    } 
    ```
  
  > 文件`ui_widget.h`里封装界面的类是`Ui_Widget`。再定义一个类`Widget`从`Ui_Widget`继承而来，并将其定义在名称空间Ui里，这样`Ui::Widget`与文件`widget.h`里的类名`Widget`同名，但是用名称空间据分开。

## 窗口的创建与初始化

再回过头来看文件widget.h里Widget类的定义:

```c++
class Widget : public QWidget
{
    Q_OBJECT

public:
    Widget(QWidget *parent = nullptr);
    ~Widget();

private:
    Ui::Widget *ui;
};
```

它定义了`Ui::widget`类型的指针变量`ui`，通过这个变量，就可以访问到界面上的所有组件。

`widget`的构造函数如下：

```c++
Widget::Widget(QWidget *parent)
    : QWidget(parent)
    , ui(new Ui::Widget)
{
    ui->setupUi(this);
}
```

在构造函数中调用`ui->setupUi(this)`进行窗口界面初始化，包括创建所有界面组件、设置属性、设置信号与槽的关联。`this`是`Widget`实例对象，它是一个`QWidget`窗口，是`setupUi(this)`中创建的所有界面组件的父容器。
